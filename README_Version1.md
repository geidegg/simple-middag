# Middags-swiper (prototyp)

En enkel prototyp av en app där användaren swipar mellan maträtter för att få idéer till middag. Funktionenalitet i prototypen:
- Filtrera på kök (cuisine) och subkategorier (t.ex. "träningsinriktad", "billigt").
- Swipe höger för att visa receptdetaljer.
- Klicka på en ingrediens i receptvyn för att se en mockad prisjämförelse över olika butiker.

Installation
1. Installera beroenden:
   - npm install
2. Starta Expo:
   - npm start

Nästa steg / förbättringsförslag
- Koppla receptlänkar och ingredienspriser till riktiga API:er (t.ex. butikers öppna API eller web scraping med tillstånd).
- Lägga till sparade favoriter och möjlighet att skapa inköpslista från valda rätter.
- Förbättra UI/UX, animera swipe, visa bilder per rätt.
- Implementera användarprofiler och personliga rekommendationer.

Språk: TypeScript + React Native (Expo)
